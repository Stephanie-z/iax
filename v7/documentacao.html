Configuração e hospedagem no GitHub

1. Criando uma Conta no GitHub

Para começar a hospedar os códigos do seu site no GitHub, o primeiro passo é criar uma conta. Siga os passos abaixo:

  1. Acesse o site oficial do GitHub: https://github.com.
  2. Clique na opção "Sign Up" (Inscrever-se) para criar uma nova conta.
  3. Preencha o formulário de cadastro com as informações solicitadas:
    • Email: Insira um endereço de e-mail válido.
    • Senha: Crie uma senha segura.
    • Username: Escolha um nome de usuário único, que será usado para identificá-lo no GitHub.
  4. Você receberá um código de confirmação no seu e-mail. Insira esse código para confirmar seu cadastro.
  5. Após a confirmação, você estará pronto para usar sua conta do GitHub.

2. Criando um Repositório no GitHub

Após criar a conta no GitHub, é hora de criar um novo repositório, onde serão armazenados os códigos do seu site. Siga os passos abaixo:

  1. No site do GitHub, clique no ícone de "+" no canto superior direito e selecione "New Repository".
  2. Preencha as seguintes informações:
    • Nome do Repositório: por exemplo, "nosso-site".
    • Descrição: opcional, mas recomendável.
    • Visibilidade: selecione "Public".
    • Marque a opção "Initialize with a README".
  3. Clique em "Create Repository".

3. Hospedando Arquivos e Códigos das Páginas do Site no GitHub

Agora você pode adicionar arquivos ao repositório diretamente pela interface web do GitHub:

Organizando os Arquivos por Pastas

  1. Clique em "Add file" > "Create new file".
  2. Para criar a pasta HTML e o arquivo index.html: digite HTML/index.html.
  3. Adicione o conteúdo do arquivo no editor que aparece abaixo.
  4. Adicione uma descrição e clique em "Commit new file".

Repetindo o Processo

Repita os passos para outros arquivos (CSS, JavaScript, imagens, etc), organizando-os em suas respectivas pastas.
