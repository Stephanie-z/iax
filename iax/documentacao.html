Configuração e hospedagem no GitHub

1. Criando uma Conta no GitHub

Para começar a hospedar os códigos do seu site no GitHub, o primeiro passo é criar uma conta. Siga os passos abaixo:

  1. Acesse o site oficial do GitHub: https://github.com.
  2. Clique na opção "Sign Up" (Inscrever-se) para criar uma nova conta.
  3. Preencha o formulário de cadastro com as informações solicitadas:
    • Email: Insira um endereço de e-mail válido.
    • Senha: Crie uma senha segura.
    • Username: Escolha um nome de usuário único, que será usado para identificá-lo no GitHub.
  4. Após preencher o formulário, você receberá um código de confirmação no seu e-mail. Insira esse código para confirmar seu cadastro.
  5. Após a confirmação, você estará pronto para usar sua conta do GitHub.

2. Criando um Repositório no GitHub

Após criar a conta no GitHub, é hora de criar um novo repositório, onde serão armazenados os códigos do seu site. Siga os passos abaixo:

  1. No site do GitHub, no canto superior direito, clique no ícone de "+" e selecione a opção "New Repository" (Novo Repositório).
  2. Na página de criação do repositório, preencha as seguintes informações:
    • Repository Name (Nome do Repositório): Escolha um nome para o repositório, por exemplo, "nosso-site".
    • Description (Descrição): A descrição é opcional, mas é recomendável adicionar uma breve explicação sobre o objetivo do repositório, como "Repositório para armazenar os códigos do site da empresa fictícia."
    • Public (Público): Selecione a opção Público para permitir que outras pessoas vejam seu repositório.
    • Initialize this repository with a README (Inicializar este repositório com um arquivo README): Marque essa opção para criar um arquivo README.md, onde você pode adicionar informações úteis sobre o projeto.
  3. Clique no botão Create Repository (Criar Repositório) para finalizar.

3. Hospedando Arquivos e Códigos das Páginas do Site no GitHub via Interface Web

Após criar o repositório, você pode começar a adicionar e organizar os arquivos de código do seu site diretamente pelo GitHub. Para isso, siga os passos abaixo:

3.1. Organizando os Arquivos por Pastas

Você pode criar pastas para organizar os arquivos de acordo com o tipo (HTML, CSS, JavaScript, etc.). A seguir, explicamos como criar as pastas e arquivos diretamente no GitHub:

  1. No seu repositório recém-criado, clique na opção "Add file" e depois selecione "Create new file" (Criar novo arquivo).
  2. No campo "Name your file" (Nomeie seu arquivo), digite o nome da pasta e o arquivo que deseja criar. Por exemplo, para criar a pasta HTML e o arquivo index.html dentro dela. Faça HTML e /index.html. O GitHub criará automaticamente a pasta HTML e o arquivo index.html dentro dela.
  3. Após inserir o nome do arquivo, veja o editor de código logo abaixo. Adicione o código HTML ou o conteúdo necessário no arquivo.
  4. Se necessário, adicione um comentário para descrever a alteração ou o conteúdo do arquivo.

3.3. Repetindo o Processo para Outros Arquivos

Repita esse processo para todos os arquivos do seu site, como HTML, CSS, JavaScript, imagens, entre outros, sempre organizando-os em pastas adequadas.
